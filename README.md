# VPL_HAAR_Algorithm
Dieses Projekt zielt darauf ab, auf Basis einer Gesichts- und Emotionserkennung ein Set aus
Bildern auszuwerten, um herauszufinden, ob auf den Bildern eine lächelnde Person zu sehen
ist. Angefangen von Fernsehshows, wie L.O.L. (Last one laughing), über Verhaltensanalysen bis
hin zu verschiedenen Mensch-Maschine-Interaktionen, besitzt die Erkennung eines menschlichen
Lächelns ein breites Einsatzspektrum und kann als Bestandteil diverser Weiterentwicklungen ge-
nutzt werden.
Herausforderungen bilden hierbei verschiedene Perspektiven, eine zuverlässige Erkennung der
Gesichtsmerkmale bei verschiedenen Personen und äußere Faktoren, wie Lichtverhältnisse.
2.1 Ziel des Projekts
Ziel des Projekts ist eine möglichst zuverlässige, automatische Erkennung von lächelnden Ge-
sichtern auf Basis von Bilddaten.
Langfristig kann das System zu einer Anwendung in Echtzeit weiterentwickelt werden.
2.2 Geplante Verarbeitungsschritte
Der aktuelle Stand sieht folgende Verarbeitungsschritte vor, die aufeinander aufbauen:
1. Einlesen der Bilddaten
2. Gesichtsdetektion
3. Erkennung von Gesichtsmerkmalen
4. Analyse der Mundposition
5. Klassifikation des Gesichts
